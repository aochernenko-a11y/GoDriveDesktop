# GoDriveDesktop

GoDriveDesktop — desktop-застосунок для автоматизації роботи каси автозаправної станції. Програма дозволяє касиру продавати пальне та супутні товари, формувати список покупки, застосовувати акції, підтверджувати продажі та переглядати дані зміни. Адміністратор може керувати товарами, пальним, працівниками, акціями та формувати звіти.

## Основні можливості

- авторизація користувачів;
- розмежування доступу для ролей **Адміністратор** і **Касир**;
- продаж пального через паливороздавальні колонки;
- продаж супутніх товарів магазину АЗС;
- формування списку покупки;
- застосування акцій та знижок;
- збереження продажів у базі даних;
- перегляд історії продажів і формування звітів;
- керування товарами, пальним, працівниками та акціями.

## Технології

- C#;
- .NET 9;
- Windows Forms;
- MySQL;
- MySql.Data.

## Вимоги для запуску

Перед запуском потрібно встановити:

- Windows 10/11;
- Visual Studio 2022;
- .NET 9 SDK;
- MySQL Server або XAMPP з модулем MySQL;
- MySQL Workbench або phpMyAdmin для імпорту бази даних.

## Завантаження проєкту

Проєкт можна завантажити з GitHub двома способами.

Перший спосіб — через ZIP-архів:

1. Відкрити сторінку репозиторію GitHub.
2. Натиснути кнопку **Code**.
3. Обрати **Download ZIP**.
4. Розпакувати архів у зручну папку.

Другий спосіб — через Git:

```bash
git clone https://github.com/aochernenko-a11y/GoDriveDesktop.git
```

## Встановлення бази даних

1. Запустити MySQL Server або XAMPP.
2. Відкрити MySQL Workbench або phpMyAdmin.
3. Виконати SQL-скрипт `БД.sql` для створення бази даних і таблиць.
4. Виконати SQL-скрипт `Заповнювач.sql` для додавання початкових даних.

Після імпорту буде створена база даних `GoDriveDB`.

## Налаштування підключення до бази даних

У файлі `DBConnection.cs` потрібно перевірити рядок підключення:

```csharp
"server=127.0.0.1;port=3306;database=GoDriveDB;user=root;password=Qq151190;charset=utf8mb4;"
```


## Запуск програми через Visual Studio

1. Відкрити файл `GoDriveDesktop.sln` у Visual Studio 2022.
2. Дочекатися відновлення NuGet-пакетів.
3. Переконатися, що MySQL запущений.
4. Натиснути **Start** або клавішу **F5**.

## Структура проєкту

```text
GoDriveDesktop/
├── AdminMainForm.cs          # форма адміністратора
├── CashierMainForm.cs        # форма касира
├── LoginForm.cs              # форма авторизації
├── DBConnection.cs           # підключення до бази даних
├── FullScreenForm.cs         # базова форма для повноекранного режиму
├── RoundedButton.cs          # власний клас кнопки
├── RoundedPanel.cs           # власний клас панелі
├── Models/                   # класи моделей
├── Resources/                # графічні ресурси
└── GoDriveDesktop.csproj     # файл проєкту
```

## Примітка

Перед запуском програми база даних повинна бути створена, а параметри підключення у файлі `DBConnection.cs` мають відповідати налаштуванням локального MySQL-сервера.
